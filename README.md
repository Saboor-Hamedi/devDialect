## ğŸ“¦ Getting Started

### Prerequisites

-   PHP 8.2+
-   Composer
-   Node.js & NPM

### Installation

1. **Clone the repository**

    ```bash
    git clone https://github.com/yourusername/devdialect.git
    cd devdialect
    ```

2. **Install dependencies**

    ```bash
    composer install
    npm install
    ```

3. **Environment Setup**

    ```bash
    cp .env.example .env
    php artisan key:generate
    ```

4. **Initialize Database**

    ```bash
    php artisan migrate
    ```

5. **Start Dev Servers**
    ```bash
    npm run dev
    php artisan serve
    ```
